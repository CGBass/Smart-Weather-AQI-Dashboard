🌦 Smart Weather & AQI Dashboard

A modern premium Weather & Air Quality (AQI) Dashboard built using HTML, CSS, and JavaScript.

This application provides real-time weather data, AQI visualization, and dynamic background updates using OpenWeather API and Unsplash images.

🚀 Features

- 📍 Auto Location Detection
- 🔍 City Search Functionality
- 🌡 Real-time Temperature & Weather Info
- 🌫 Live Air Quality Index (AQI)
- 🎯 Animated AQI Circular Meter
- 🌅 Accurate Sunrise & Sunset (Timezone Fixed)
- 💨 Wind Speed, Humidity, Pressure & UV Index
- 🖼 Dynamic Background Changes on Every Load & Search
- ✨ Modern Glassmorphism Premium UI
- 🎨 Responsive & Smooth Design

 🛠 Technologies Used

- HTML5
- CSS3 (Glassmorphism Design)
- JavaScript (ES6)
- OpenWeather API
- Unsplash Image CDN

This project is open-source and free to use.
